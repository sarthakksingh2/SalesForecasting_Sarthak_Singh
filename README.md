# Sales Forecasting and Demand Prediction System

## Overview
This Streamlit application was developed as part of a Data Science Internship Project. It analyzes historical Superstore sales data and provides sales forecasting, anomaly detection, and product demand segmentation.

## Features
- Sales Overview Dashboard
- XGBoost Sales Forecasting
- Isolation Forest Anomaly Detection
- Product Demand Segmentation using K-Means Clustering

## Technologies Used
- Python
- Streamlit
- Pandas
- Plotly
- Scikit-Learn
- XGBoost
- Statsmodels
- Prophet
